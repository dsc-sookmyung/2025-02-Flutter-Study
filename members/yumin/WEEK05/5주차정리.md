# 😊 My Profile

## 📚 SingleChildScrollView

- SingleChildScrollView는 한 방향으로 스크롤 가능한 화면을 만드는 위젯.
- 화면 크기보다 내용이 길 때 오버플로우(overflow) 오류 없이 자연스럽게 스크롤됨.
- 보통 Column과 함께 사용해 세로로 여러 위젯을 배치할 때 자주 쓰임.

## 🫧 crossAxisAlignment

- crossAxisAlignment는 Column이나 Row 내에서 교차 축(가로/세로) 방향의 정렬을 지정함.
- crossAxisAlignment.start의 경우 자식들을 왼쪽(또는 위쪽)에 정렬함.
- 이를 통해 텍스트나 카드의 정렬 방향을 조절 가능.

## 📁 파일 분리

- DescriptionCard를 별도 파일(description_card.dart)로 분리함으로써 코드의 재사용성과 가독성 높임.
- UI 구조(화면)와 위젯 로직(설명 카드)을 분리해 유지보수 및 수정이 용이함
- 다른 페이지에서도 같은 카드 위젯을 import만으로 재사용 가능.

## ☁️ 주요 위젯

- MaterialApp: 앱 전체의 기본 틀, 테마와 라우팅을 담당.
- Scaffold: 앱의 기본 레이아웃 구조(AppBar, Body 등 포함).
- AppBar: 상단에 제목, 메뉴 등을 표시하는 영역.
- Column: 여러 위젯을 세로로 나열.
- Padding: 위젯 주위에 여백을 줌.
- Container: 박스 형태의 기본 위젯으로 색상, 크기, 테두리 설정 가능.
- Text: 문자열 표시용 위젯.
- Divider: 구분선 표시용 위젯.
- SizedBox: 일정한 크기의 간격을 만듦.
