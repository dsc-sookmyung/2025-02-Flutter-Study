- 동기 프로그래밍 : 한 작업이 끝나야 다음 작업 실행 가능
 
- **async + await** : 비동기 프로그래밍
   - 시간이 걸리는 작업을 수행할 수 있는 방법
    - 백그라운드에서 작업 가능

```dart
  void getLocation() async{

    final LocationSettings locationSettings = LocationSettings(
      accuracy: LocationAccuracy.low,
      distanceFilter: 100,
    );

    Position position = await Geolocator.getCurrentPosition(locationSettings: locationSettings);
    print(position);
  }
```

- **StatelessWidget** : 변화를 원할때마다 파괴하고 새로 만들어야함
- **StatefulWidget** : 상태개체를 이용해 상태를 추적할 수 있음

- **LifecycleWidget**
  - initState() : 초기화하고 호출될 때 나타남
  - deactivate() : 파괴될 때 나타남
  - build() : 빌드될 때 나타남
