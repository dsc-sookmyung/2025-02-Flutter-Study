# 전하은 - 5주차 정리
### 나만의 프로필 카드 만들기

* import
```dart
//import해야 다른 디렉토리의 요소 사용 할 수 있다.
import 'Widget/description_card.dart';
```

* SingleChildScrollView
```dart
//column은 스크롤 안됨 -> 오버플로우 시 SingleChildScrollView를 사용해 스크롤
body: SingleChildScrollView()
```

* AppBar
```dart
title: const Text(
  '저를 소개합니다',
  //TextStyle(글꼴, 색, 굵기, 크기 등 폰트 설정 가능)
),
centerTitle: true, // AppBar 텍스트 가운데 정렬
```

* Padding / SizedBox
```dart
Padding(
  padding:EdgeInsets.only(top: 30.0),
),
SizedBox(height: 50,), // 위젯 간 간격 설정
```

* CircleAvatar
```dart
CircleAvatar(
  backgroundImage: AssetImage('assets/my_profile.gif'),
  radius: 80, // 원 크기 설정
);
```

* DescriptionCard 분리
```dart
//디렉토리와 다트파일을 새로 만들어 클래스를 main파일과 분리 할 수 있다.
class DescriptionCard extends StatelessWidget { ... }
```

*  Column 구조
```dart
//sizedbox를 사용해도 되지만 컨테이너의 패딩 온리 바텀을 설정해도 간격을 띄울 수 있다.
padding: EdgeInsets.only(bottom: 20.0);

//color가 null이면 F3F8FF를 컬러로 함
color: color ?? Color(0xffF3F8FF);

//column은 세로, row는 가로
//crossAxisAlignment는 세로축을 설정, start로 왼쪽정렬
crossAxisAlignment: CrossAxisAlignment.start,
```
